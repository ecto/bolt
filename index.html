Greetings, Internet visitor! You have stumbled upon Bolt, a library intended to make life easy.

Getting Started

npm install bolt

Make a new Bolt

var Bolt = require('bolt');
var $ = new Bolt();

Opinions

Web development with node should be fast and easy.

Modularity is one of the best features of node. Several modules have emerged that are both ubiquitous requirements for most web projects, and higher quality than their counterparts. Bolt takes advantage of these modules by including them by default: users are not forced to hunt down the gems, but rather given them for free.

This approach is not for everyone. If the modules that are included with Bolt are not satisfactory, they can be overridden.

Responding to requests

$.get('/', function (req, res) {
  res.send('Hello, world.');
});

Dynamic routing

var users = [
  johnny: {
    type: 'dog',
    name: 'Johnny'
  }
];

$.get('/:user', function (req, res) {
  res.render('Hello, world.', {
    user: users[req.params.user]
  });
});

Making requests

Bolt uses mikeal's request module. It's awesome! Here is a simple proxy:

$.get('/proxy', function (req, res) {
  $.request('http://google.com', function (err, response, body) {
    res.send(body);
  });
});

Simple streams!

$.get('/:query', function (req, res) {
  var q = encodeURIComponent(req.params.query);
  request.get('http://google.com/?q=' + q).pipe(res);
});
