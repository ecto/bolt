bolt

Send messages to any node.js process, anywhere on the Internet.

View on GitHub

bolt intercepts EventEmitter emissions and shoots them over Redis pubsub. All bolt nodes will receive these events and trigger emissions themselves. This allows for simple, advanced inter-process message passing around the same system or between systems.

install

npm install bolt

usage

var bolt = require('bolt');

var mesh = new bolt.Node({
  host: 'myserver.com'
  auth: 'my redis password'
});

mesh.start();

mesh.on('ping', function(){
  mesh.emit('pong');
});
